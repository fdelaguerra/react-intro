React test app

This project is part of an independent study where I began learning React from scratch.

This project used create-react-app to get started and JSX to build out on a concept I had for a portfolio page. I used it to continue my learning from the UF Coding and Web Apps course. That course's professor at the time did not have experience with React, but mentored me as best she could throughout the process.
